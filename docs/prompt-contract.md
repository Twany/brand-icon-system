# Prompt Contract

This document describes the expected inputs and outputs for each stage. Host-specific syntax may vary, but the stages should remain the same.

## Brief

Input: product theme, brand name, existing logo, or rough direction.

Output:

- brand name;
- product category;
- audience;
- tone;
- design tension;
- visual core;
- candidate territories;
- avoid list;
- color system;
- production notes.

## Explore

Input: brief.

Output: one 4x6 black-and-white exploration board as PNG, plus split candidate files.

Prompt shape:

```text
Create a minimalist black-and-white logo exploration board for [brand/theme].
4 columns x 6 rows, 24 distinct concepts, pure white background, black flat graphics only.
Each cell is independently centered with generous whitespace. Explore different directions:
negative space, bold geometric mass, modular mark, letter fusion, visual pun, product action,
calm symbol, rhythmic repetition, custom wordmark, and abstract emblem.
The work should feel like a restrained identity-studio exploration archive.
No color, gradients, shadows, 3D, mockups, texture, labels, numbering, UI, watermark, or cartoon illustration.
```

## Refine

Input: selected candidate number or rough source mark.

Output: one to four isolated PNG mark options.

Prompt shape:

```text
Minimal premium app logo mark for [brand/theme], based on direction #[N] from the exploration board.
No text, simple rounded geometric symbol, strong negative space, crisp solid black silhouette on pure white background.
Make it readable at 32px, with natural proportions and no tiny interior fragments.
High-resolution PNG, no shadow, no gradient, no mockup, no texture, no background pattern.
```

## Produce

Input: approved source mark PNG, brand name, color system, output directory.

Output: full PNG asset system generated by `build_icon_system.py`.

## Audit

Input: output directory.

Output: pass/fail review with exact rerun instructions when needed.
