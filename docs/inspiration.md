# Inspiration

The exploration stage is inspired by the public `fucha1122/minimalist-bw-logo-skill` project.

The borrowed idea is procedural, not textual:

- start with many black-and-white concepts;
- prioritize broad visual variety;
- choose a direction before production;
- evaluate marks by small-size clarity and simplicity.

This repository does not copy that project's prompt text or implementation. It adds:

- an explicit staged suite;
- a user-selection interaction contract;
- raster PNG production scripts;
- theme-color asset exports;
- final output audit.
